# Создание веб-приложения с использованием React + Redux + Redux-Saga
Цель создания приложения - демонстрация всех элементов Redux/Redux-Saga.

## Состав приложения:
1. Компонент формы для создания постов.
2. Компонент списка синхронных постов.
3. Компонент списка асинхронных постов из сервера (использование асинхронных событий и лоадера).
### `Для запуска приложения необходимо в терминале выполнить команду npm run start (npm start)`

**Для сборки в продакшн режим в термине следует выполнить команду npm run build**
